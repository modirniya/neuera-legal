2. Service Description

Odo is a free mileage tracking application for iOS and Android devices that allows you to:

  • Manually log trips with start and end odometer readings
  • Capture optional location snapshots for geographic context
  • Track vehicle-related expenses (fuel, tolls, parking, maintenance, car wash, other)
  • Categorize trips as business or personal
  • Calculate mileage deductions using IRS rates
  • Manage multiple vehicles
  • View monthly statistics and summaries
  • Export data to CSV and PDF formats (includes location data)
  • Import trip and expense data from CSV files

Key Features:

  • Local Storage: All data stays on your device
  • Manual Entry: All mileage calculated from odometer entries you provide
  • Optional Location: Capture GPS snapshots (not tracking) for context
  • Completely Free: No purchases, subscriptions, or ads
