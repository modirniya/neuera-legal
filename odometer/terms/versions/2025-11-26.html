2. SERVICE DESCRIPTION

2.1 What Odometer Provides

Odometer is a free, offline mileage tracking application for Android devices that allows you to:

  • Manually log trips with start and end odometer readings
  • Track vehicle-related expenses (fuel, tolls, parking, maintenance, car wash, other)
  • Categorize trips as business or personal
  • Calculate mileage deductions using IRS rates
  • Manage multiple vehicles
  • View monthly statistics and summaries
  • Export data to CSV and PDF formats
  • Import trip and expense data from CSV files

2.2 Offline-Only Operation

Odometer operates entirely offline:

  • No internet connection required or used
  • No cloud sync or backup features
  • No user accounts or registration
  • All data stored locally on your device

2.3 Free Service

Odometer is completely free:

  • No purchase required
  • No in-app purchases
  • No premium features or subscriptions
  • No advertisements
  • All features available to all users

2.4 No GPS Tracking

Despite being a mileage tracking app, Odometer does not use GPS or location services. All mileage is calculated from odometer readings you manually enter. This is a deliberate privacy-first design choice.
